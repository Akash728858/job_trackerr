# Job Notification Tracker

## Run

**Option 1 — Double-click**  
Open `index.html` in your browser. Works with `file://`.

**Option 2 — Local server**  
```bash
npx serve job-tracker
```
Open: **http://localhost:3000/**

## URLs

- `#/` or `#` — Home
- `#/dashboard` — Dashboard
- `#/saved` — Saved
- `#/digest` — Digest
- `#/settings` — Settings
- `#/proof` — Proof

Hash routing works everywhere; no server config needed.
